## 为什么要跑步？

作为程序员，我们每天久坐 8 小以上：

- 🦴 颈椎、腰椎问题
- 👀 视力下降
- 😴 精力不足
- 🧠 思维迟钝

**跑步**是门槛最低、效果最好的运动之一。

<!--more/>

## 从 0 到 10 公里的计划

### 第 1-2 周：走跑交替

```
快走 5 分钟 → 慢跑 1 分钟 → 快走 5 分钟
重复 3-4 组
```

### 第 3-4 周：增加跑步比例

```
慢跑 3 分钟 → 快走 2 分钟
重复 5 组
```

### 第 5-6 周：连续跑步

```
慢跑 15-20 分钟，不停歇
```

### 第 7-8 周：突破 5 公里

```
慢跑 30 分钟，约 5 公里
```

### 第 9-12 周：冲击 10 公里

```
每周跑 3 次，每次 40-60 分钟
```

## 装备建议

| 装备 | 建议 | 预算 |
|------|------|------|
| 跑鞋 | 亚瑟士/美津浓/李宁 | ¥300-800 |
| 运动手表 | 小米/华为 | ¥200-1000 |
| 运动耳机 | 韶音骨传导 | ¥500-1200 |
| 速干衣 | 迪卡侬就够 | ¥50-200 |

## 注意事项

1. **热身** — 跑前动态拉伸 5 分钟
2. **配速** — 能边跑边聊天的速度就对了
3. **频率** — 每周 3 次，休息日穿插
4. **恢复** — 跑后静态拉伸 10 分钟
